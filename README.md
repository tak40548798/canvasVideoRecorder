## Web Camera 錄製工具

透過 Media Capture and Streams API 存取使用者的 Web Camera

讓使用者可以對 Camera 畫面放大、縮小、拖放等....

並且還在 Camera 畫面上疊加了 Canvas 

可以透過 Canvas 繪製內容、擦除內容、上一步、下一步...

使用者在 Canvas 上所畫的圖像紀錄和 Camera 畫面會被一起錄製起來
